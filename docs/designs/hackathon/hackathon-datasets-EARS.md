# Hackathon Datasets - EARS

**Parent LLD**: ./LLD.md

## Dataset Domain

- **HACK-DATASET-001**: The system shall provide at least 3-4 publicly available biology or chemistry datasets.
- **HACK-DATASET-002**: Each dataset shall be sourced from the internet and accessible without login or API key.
- **HACK-DATASET-003**: Each dataset shall be under 50MB in size.
- **HACK-DATASET-004**: Each dataset shall be in a standard format: CSV, TSV, JSON, or Parquet.

## Dataset Documentation

- **HACK-DATASET-010**: The system shall provide a catalog entry for each dataset containing: name, source URL, format, size, domain, quick start description, and optional suggested questions.
- **HACK-DATASET-011**: When a participant selects a provided dataset, the system shall make the source URL and quick start instructions available.

## Dataset Sources

- **HACK-DATASET-020**: The system shall curate datasets from sources including: Kaggle, UCI Machine Learning Repository, PDB, NCBI/GEO, ChEMBL, PubChem, and the instructor's existing science dataset collection.

## Participant Data Options

- **HACK-DATASET-030**: The system shall allow participants to use either a provided dataset or their own data.
- **HACK-DATASET-031**: If a participant brings their own data, then the system shall warn them to ensure it is ready to load in a notebook before the workshop starts.
- **HACK-DATASET-032**: Where a participant brings their own data, the system shall encourage using data from public source URLs rather than local files.
